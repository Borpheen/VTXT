# VTXT v3.0.0 Release Notes

## Initial VTXT Release

### Included

- Dual independent countdown timers
- Always-on-top overlay pill
- Overlay visible/hidden toggle
- Overlay lock/unlock toggle
- Overlay size and opacity controls
- Built-in alarm sounds
- Custom audio file support
- Configurable keyboard and mouse-button hotkeys
- Settings persistence under `%LOCALAPPDATA%\VTXT`
- Windows installer built with Inno Setup

### Installer

Release installer:

```text
VTXT_Setup_v3.0.0.exe
```

### Notes

VTXT is a normal Windows desktop timer and overlay application. It does not inject input, remap controls, modify game files, read game memory, or interact with Battlefield 6 directly.
